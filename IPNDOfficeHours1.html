IPND Office Hours : Understanding div, span, id and class

IDLE

  1. Divs are used to control layout.
  2. Spans are used for decoration.
  3. Divs are used to group things together.
  4. Spans should be used for anything else inline.
  5. You can specify width of your div (eg columns of text)
  6. Ids are used to identify element in the webpage.
  7. Ids similar to each other can use a Class.
  8. class supersedes span
  9. Classes should be used to style
  10. Percent sign is the total pixels
  11. Id vs Class: Class is like barcode.
  12. Code-folding : can look at your code and fold parts not interested in seeing.
