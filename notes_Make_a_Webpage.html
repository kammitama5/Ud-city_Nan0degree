Focus Checkpoint

The Important points are :

  • HTML and CSS are both 'languages'.
  • HTML controls the 'structure' of a web page.
  • CSS controls the 'style' of a page (how it looks).
  • When programmers talk about the 'DOM' (Document Object Mode), they are talking about the tree structure of a page.
  • Tree Structure Wikipedia reference Page

HTML-CSS-DOM

  • HyperText Markup Language (HTML): The standard markup language used to create web pages.
  • Cascading Style Sheets (CSS): Style sheet language used for describing the look and formatting of a document written in a markup language.
  • Document Object Model (DOM): a cross-platform and language-independent convention for representing and interacting with objects in HTML (and other markup languages). The nodes of every document are organized in a tree structure, called the DOM tree.
